# booksim2 目录总结

## 功能概述
`booksim2/` 目录是BookSim2网络仿真器的集成目录，BookSim2是一个周期精确的网络-on-chip（NoC）仿真器，为RapidChiplet提供详细的性能仿真能力。

## 目录结构

### LICENSE.md
**BookSim2许可证文件**
- **内容**: BookSim2项目的开源许可证条款
- **用途**: 定义软件使用和分发的法律条款
- **重要性**: 确保合规使用和贡献指导

### README.md
**BookSim2说明文档**
- **内容**: BookSim2的基本介绍和使用说明
- **信息**: 编译方法、使用示例、配置说明
- **用途**: 帮助用户理解和使用BookSim2

### src/
**源代码目录**
- **功能**: 包含BookSim2的所有C++源代码
- **编译目标**: 生成可执行的BookSim2仿真器
- **集成方式**: 通过make命令进行编译

## BookSim2在RapidChiplet中的作用

### 周期精确仿真
- **精确性**: 提供周期级的精确网络仿真
- **详细分析**: 深入分析网络性能特征
- **验证工具**: 验证RapidChiplet快速分析的结果
- **基准对比**: 为性能评估提供基准

### 仿真流程集成
1. **配置转换**: RapidChiplet自动生成BookSim配置
2. **拓扑导出**: 网络拓扑转换为BookSim格式
3. **流量转换**: 流量模式转换为BookSim格式
4. **仿真执行**: 自动运行BookSim仿真
5. **结果收集**: 收集和分析仿真结果

### 性能指标提供
- **延迟分析**: 详细的包延迟分析
- **吞吐量分析**: 网络吞吐量性能
- **拥塞分析**: 网络拥塞特征
- **路由效率**: 路由算法效率分析

## 编译和使用

### 编译过程
