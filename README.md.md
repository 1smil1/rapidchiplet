### 安装和设置指南

#### 基础安装步骤
1. **代码获取**: Git克隆仓库
2. **依赖安装**: Python包的pip安装
3. **BookSim2编译**: C++仿真器的构建
4. **Netrace编译**: 轨迹处理工具的构建

#### 详细命令
```bash
# 克隆仓库
git clone https://github.com/spcl/rapidchiplet.git

# 安装Python依赖
cd rc
pip install -r requirements.txt

# 编译BookSim2
cd booksim2/src
make
cd ../../

# 编译Netrace
cd netrace
gcc export_trace.c netrace.c -o export_trace
cd ../
```

### RapidChiplet核心功能

#### 输入系统
- **配置文件**: 基于JSON的设计配置
- **示例文件**: `inputs/` 目录下的示例
- **模块化设计**: 分离的设计组件配置

#### 输入生成工具
1. **路由生成**: `generate_routing.py`
   - 支持SPLIF和SPTMR路由算法
   - 自动路由表计算

2. **流量生成**: `generate_traffic.py`
   - 多种合成流量模式
   - 支持参数化配置

#### 核心执行
- **基本命令**: `python3 rapidchiplet.py -df <design> -rf <results> [options]`
- **性能指标**: 面积、功耗、延迟、吞吐量等
- **可选指标**: 通过命令行标志控制

#### 周期精确仿真
- **启动命令**: 添加 `-bs` 标志
